# AllMusic | 全服点歌
---
## 简介
可以使用指令点歌并且让所有装了对应mod的玩家都听到你点的歌
## 指令
    /music [ID] — 点歌
    /music help — 获取帮助
    /music vote — 投票切歌
    /music stop — 停止播放这首歌
    /music nomusic — 不再参与点歌
    /music list — 获取点歌列表
    /music search [歌名]  — 搜索歌曲
    /music select [序号] — 选择搜索的歌曲
    /music nextpage — 搜索下一页
    /music lastpage — 搜索上一页
